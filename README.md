## 💻 Usage

### Navigation
Click any menu item in the sidebar:
- 🏠 Dashboard
- 🛒 Orders
- 📦 Products
- 📊 Analytics
- 👥 Customers
- 🚚 Delivery
- ⚙️ Settings

### Interactions
- **Click KPI cards** → View details
- **Click orders** → See order information
- **Click products** → Product details
- **Click charts** → Data points
- **Click map markers** → Location info

### Keyboard Shortcuts
- `Ctrl/Cmd + P` → Print dashboard
- `Ctrl/Cmd + E` → Export data
- `Esc` → Close sidebar (mobile)

## 🎨 Customization

### Change Colors
Edit `css/styles.css`:
```css
:root {
    --primary-color: #F8CB46;    /* Yellow */
    --secondary-color: #0C831F;  /* Green */
    --accent-color: #E23744;     /* Red */
}
```

### Update Data
Edit `js/data.js`:
```javascript
const dashboardData = {
    kpis: {
        totalRevenue: 1250000,
        totalOrders: 8543,
        // ... your data
    }
};
```

### Add Features
1. Add HTML in `blinkit-dashboard.html`
2. Add styles in `css/styles.css`
3. Add logic in `js/main.js`
