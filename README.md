## 💻 Usage

### Navigation
Click any menu item in the sidebar:
- 🏠 Dashboard
- 🛒 Orders
- 📦 Products
- 📊 Analytics
- 👥 Customers
- 🚚 Delivery
- ⚙️ Settings

### Interactions
- **Click KPI cards** → View details
- **Click orders** → See order information
- **Click products** → Product details
- **Click charts** → Data points
- **Click map markers** → Location info

### Keyboard Shortcuts
- `Ctrl/Cmd + P` → Print dashboard
- `Ctrl/Cmd + E` → Export data
- `Esc` → Close sidebar (mobile)

## 📱 Responsive Design

| Device | Screen Size | Layout |
|--------|-------------|--------|
| **Mobile** | < 768px | Single column, hamburger menu |

2. Add styles in `css/styles.css`
3. Add logic in `js/main.js`
